Matt Triplett 18-May-18 12:12 PM
What's stopping you? It seems like a great RL/DL introduction project! Aside from the unique issues of getting your environment up and running the whole structure is pretty basic and open, which makes sense because it was intended to be the jumping-off point for a sort of hackathon. Only the last couple sections of the page are the difficult, mental-effort-required sections, and there is a lot there, so I see how it can be intimidating. Breaking down the really tough subsections (Reward Shaping, Curr. Learning, Heir. RL) as their own independent projects may help you if you're feeling overwhelmed. The post has a few links to some resources, and the write-up for the event itself has a few great resources as well: https://njustesen.com/2018/01/30/report-from-starcraft-ai-workshop-at-itu-january-2018/ The SC1 Slideshow is an interesting history lesson, and while it provides some practical examples of ML concepts in relation to SC, it seems like their approach to deep learning was very different than the one used in the main article, so don't sweat the tech details too much unless it sounds more interesting and like you'd rather work on that. The next presentation is a pretty good overview of ML/DL/RL concepts, and it was designed to cover the basics of what you need to know to get started on the project here. Despite the notation, it's all pretty straightforward. The rest of the page's sections are write-ups of the projects that other people worked on, so if you're feeling stuck, don't have strong ideas, or don't want to start from scratch, a few of these seem like nice jumping off points, or good case studies for how RL concepts can apply to SC2. Finally, and most importantly, the goal of the community around this project was to create a "grassroots" interest group of people who want to work on it, so I'm sure that if you reach out, they'd be happy to help (or at least point you to some resources).
PS this repo helps with some specific RL/RL+DL model/algorithm guides: https://github.com/jonasbusk/reinforcement_learning
jonasbusk/reinforcement_learning
reinforcement_learning - Introduction to deep reinforcement learning with examples.
